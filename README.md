## download dependencies 
"uv" is used for fast loading and convenience.

```bash
pip install -r requirements.txt
```

## run the project

```bash
# Activation of virtual space
venv\Scripts\activate 
```

### Atom
Project launch
```bash
py src/Atom/main.py
```

### Terminal Render
Project launch
```bash
py src/TerminalRender/main.py
```
